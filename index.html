A Comparative Behavioral Measure of Value Alignment

EigenBench is a black-box framework for quantifying value alignment across language models. Given a model ensemble, a constitution, and a scenario dataset, models judge each other's responses in pairwise comparisons — fitted with a Bradley-Terry-Davison model and aggregated via EigenTrust into consensus alignment scores.
